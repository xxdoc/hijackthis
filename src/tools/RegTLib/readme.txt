Type Library (.tlb) registration tool by Alex Dragokas.

Using:

Regtlib.exe [Path\file.tlb] [/admin]

/admin - [optional key], to register tlb under HKLM (system wide) hive.

-----------

HiJackThis note:

This program is not included in HiJackThis Fork resources.
However, it is used as a helper to open HJT project correctly on the new developer's machine.
